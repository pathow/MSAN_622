Breakdown of Movies from GGplot2

Data Preprocessing

My main method of data processing was within R. To reduce the dimensionality to something more manageable for Tableau, I subsetted the data based on only taking those rows that had an MPAA rating. This made the entire dataset only about ~5000 rows of movies.

Functionality

C-Level: This is a Tableau dashboard.

Discussion of Functionality

I implemented the dashboard form of three charts in Tableau, mostly based on numeric and readily quantified dimensions of the data. The first chart is a box-and-whisker plot of user ratings broken down by MPAA ratings. The second is a bar chart reflecting the counts of movies under each MPAA rating. The bottom chart is a scatterplot showing the number of user votes against the average user rating of a movie. I explored moving on to D3 versions of these plots, but it turned out that box plots are much more difficult to implement in D3 than I had anticipated. So I'm happy with simply having the Tableau versions
